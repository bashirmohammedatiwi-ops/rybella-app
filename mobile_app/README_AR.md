# تطبيق كوزماتيك - Cosmatic

تطبيق Flutter لتجارة التجميل الفاخرة. واجهة عربية كاملة، RTL، الدينار العراقي.

## تشغيل التطبيق

1. **تثبيت الحزم:**
   ```
   flutter pub get
   ```

2. **ضبط عنوان الـ API** في `lib/core/constants/api_constants.dart`:
   - **محاكي أندرويد:** `http://10.0.2.2:8000/api/v1` (افتراضي)
   - **محاكي iOS:** `http://127.0.0.1:8000/api/v1`
   - **جهاز حقيقي:** `http://YOUR_COMPUTER_IP:8000/api/v1`

3. **تشغيل التطبيق:**
   ```
   flutter run
   ```

4. **التأكد من أن السيرفر Laravel يعمل** على المنفذ 8000.

## الشاشات

- سبلاش → تعرفه بالعلامة → الصفحة الرئيسية
- الرئيسية: بانرات، فئات، منتجات مميزة، أكثر مبيعاً
- المنتجات: قائمة مع فلترة وترتيب
- تفاصيل المنتج: معرض صور، سعر IQD، إضافة للسلة والمفضلة
- السلة والتوصيل (ضيف بدون تسجيل)
- المفضلة، تتبع الطلبات، الإعدادات
