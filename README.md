# Cosmatic – نظام تجارة إلكترونية فاخرة للتجميل

نظام متكامل لتجارة التجميل الفاخرة، واجهة عربية كاملة، سوق عراقي، وعملة الدينار العراقي.

## هيكل المشروع

```
mobile/
├── backend/          # Laravel REST API + لوحة إدارة ويب
├── mobile_app/       # تطبيق Flutter (جوال)
└── docs/             # توثيق API ونظام التصميم
```

## المتطلبات

- **Backend:** PHP 8.2+, Composer, MySQL
- **Mobile:** Flutter SDK 3.2+
- **Admin:** متصفح حديث (يُخدم من Laravel)

---

## 1) تشغيل الـ Backend (Laravel)

```bash
cd backend
cp .env.example .env
# تعديل .env: DB_DATABASE, DB_USERNAME, DB_PASSWORD, APP_URL
composer install
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan storage:link
php artisan serve
```

- **API:** `http://localhost:8000/api/v1`
- **لوحة الإدارة:** `http://localhost:8000/admin/login`  
  - افتراضي: `admin@cosmatic.iq` / `password`

---

## 2) تشغيل تطبيق Flutter

```bash
cd mobile_app
flutter pub get
```

تعديل عنوان الـ API في `lib/core/constants/api_constants.dart`:

- محاكي أندرويد: `http://10.0.2.2:8000/api/v1`
- محاكي iOS: `http://127.0.0.1:8000/api/v1`
- جهاز حقيقي: عنوان IP جهازك، مثلاً `http://192.168.1.5:8000/api/v1`

ثم:

```bash
flutter run
```

## 3) المميزات الرئيسية

### تطبيق الجوال (Flutter)
- واجهة عربية كاملة واتجاه RTL
- شاشات: سبلاش، تعريف، رئيسية، قائمة منتجات، تفاصيل منتج، سلة، دفع ضيف، تتبع طلب، المفضلة، إعدادات
- تصميم فاخر: وردي، أبيض، ذهبي، بطاقات ناعمة وظلال خفيفة
- تسوق ضيف بدون تسجيل دخول
- عرض الأسعار بالدينار العراقي (د.ع)

### لوحة الإدارة (ويب)
- عربية و RTL
- لوحة تحكم: مبيعات، طلبات اليوم، تنبيه مخزون، أفضل المنتجات مبيعاً
- إدارة المنتجات (صور متعددة، سعر، خصم، مخزون، فئة)
- إدارة الفئات (متداخلة)
- إدارة الطلبات وحالاتها: قيد الانتظار / قيد المعالجة / تم الشحن / تم التسليم / ملغي
- قسائم وكوبونات
- بانرات الصفحة الرئيسية

### الـ API (Laravel)
- REST بدون مصادقة للعميل (ضيف)
- مصادقة الجلسة للإدارة فقط
- نقاط: بانرات، فئات، منتجات (قائمة/مميزة/أكثر مبيعاً/تفاصيل)، طلبات (إنشاء/تتبع)، مفضلة، التحقق من القسيمة
